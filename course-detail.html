About this course

This comprehensive UI/UX Design course takes you from beginner to job-ready. Learn design thinking, user research, wireframing, prototyping with Figma, and conducting usability tests. You'll build a strong portfolio project throughout the course, preparing you for a successful career in product design.

  • Understand core UX principles and methodologies.
  • Master Figma for high-fidelity prototyping and collaboration.
  • Conduct user research, persona creation, and user flows.

What you'll learn

Build user-friendly interfaces from scratch.

Create engaging interactive prototypes.
